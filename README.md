# financial-results-screener
Automated financial data extraction, quarterly results analysis, and stock screening platform built using Python, Pandas, Web Scraping, and Streamlit

## Features
- Quarterly results extraction
-  net profit analysis
- Earnings growth screening
- Equity research support

## Tech Stack
- Python
- Streamlit
- Pandas
- Web Scraping
- Excel

## How to Run

pip install -r requirements.txt

streamlit run app.py
